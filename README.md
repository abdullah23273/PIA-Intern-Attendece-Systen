Project Title:
Intern Attendance Management System

About the Project:
This project is a web-based Intern Attendance Management System designed to manage attendance records for interns across different departments such as HR, Finance, IT, and Engineering. The system provides a centralized platform to record, monitor, and manage intern attendance efficiently.

Technologies Used:

ASP.NET (C#)

SQL Server

Bootstrap

HTML / CSS / JavaScript

Key Features:

Department-wise intern management (HR, Finance, IT, Engineering)

Daily attendance marking system

Attendance tracking and record management

User-friendly dashboard interface

Secure database storage

Purpose:
This project was developed as a practice and portfolio project to demonstrate ASP.NET web development, database integration, and real-world management system implementation.
